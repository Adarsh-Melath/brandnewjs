## Functions

-   Functions are blocks of code that can be executed multiple times. They are declared using the keyword “function” followed by a name and a set of parentheses.

```javascript
function greet() {
    console.log('Hello, World!');
}

greet();

function greet(name) {
    console.log('Hello, ' + name + '!');
}

greet('Adarsh Melath');
```
